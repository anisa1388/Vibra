# ویبرا - قالب لندینگ فروشگاهی HTML/CSS/JS

# معرفی
ویبرا یک قالب تک صفحه ای (لندینگ پیج) فروشگاهی برای معرفی و فروش محصولات است.
قالب کاملا "ریسپانسیو" است و در لپ‌تاپ، تبلت و انواع گوشی‌ها (کوچک، متوسط، بزرگ) بدون مشکل نمایش داده می‌شود.  

*ویژگی‌ها:*
- اسلایدر متحرک شامل سه عکس در ابتدای صفحه
- طراحی جذاب و زیبا با انیمیشن‌ها و افکت‌های جذاب
- صفحه جزئیات محصول با نمایش محصولات مشابه
- سایت فارسی و راست چین (RTL)
- فونت فارسی جذاب و خوانا

# زبان های کدنویسی
- HTML5
- CSS3
- JavaScript
